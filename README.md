# Image Encryption Tool using Python

This project implements a basic image encryption and decryption tool using pixel manipulation.

## Description
The program encrypts an image by modifying pixel RGB values using a key and decrypts it back to the original image.

## Concepts Used
- Image Processing
- Pixel Manipulation
- Encryption & Decryption
- Python Pillow Library

## Technologies Used
- Python
- Pillow Library

## How to Run
python image_encrypt.py

This project was developed as part of a cybersecurity internship task.
